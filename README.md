# music-dot

🎵 MusicDot — Projeto Final
Turma de Programação IV (2026) · ISPLH
Bem-vindo ao repositório do projecto MusicDot — o site que vais construir ao longo das 8 semanas do curso de HTML, CSS e JavaScript.

📋 O que é este projecto?
O MusicDot é uma escola online de música. Durante o curso, vais construir o site completo da empresa, semana a semana, aplicando tudo o que aprendes na apostila.

🚀 Como começar (passo a passo)
1. Aceitar a tarefa
Clica no link de convite que o professor te enviou.
O GitHub vai criar automaticamente o teu repositório pessoal.
2. Clonar o repositório no teu computador
Abre o terminal (ou o Git Bash) e escreve:
bashgit clone https://github.com/ISPLH/music-dot-SEU-NOME.git

Substitui SEU-NOME pelo teu nome de utilizador do GitHub.

3. Abrir no VS Code
bashcd music-dot-SEU-NOME
code .

💾 Como guardar o trabalho (commit e push)
Depois de cada sessão de trabalho, guarda as alterações no GitHub:
bash# 1. Ver o que alteraste
git status

# 2. Adicionar os ficheiros alterados
git add .

# 3. Fazer commit com uma mensagem descritiva
git commit -m "Semana 1: adicionar estrutura HTML da página sobre.html"

# 4. Enviar para o GitHub
git push

💡 Dica: faz commit com frequência, não só no fim. Cada commit é como um "guardar jogo" — podes sempre voltar atrás.


📬 Entrega final (Semana 8)
Na semana 8, para além de teres o projecto completo, deves abrir um Pull Request:

No GitHub, abre o teu repositório
Clica em "Contribute" → "Open pull request"
Dá o título: Entrega final — SEU NOME
Clica em "Create pull request"

O professor vai usar o Pull Request para rever o teu código e dar feedback.
